## reactionTest

간단한 **반응 속도 테스트** 프로젝트입니다.

### 개요
- 화면에 신호가 나타나면 가능한 한 빠르게 반응 시간을 측정합니다.
- 측정 결과를 기록해 평균 반응 시간을 확인할 수 있습니다.

### 실행 방법
1. 프로젝트를 실행합니다.
2. 안내 문구가 나오면 준비합니다.
3. 신호가 나타나는 즉시 입력(클릭/키 입력)합니다.
4. 결과(밀리초)를 확인합니다.

### 참고
- 조용한 환경에서 테스트하면 더 안정적인 결과를 얻을 수 있습니다.
- 여러 번 측정한 뒤 평균값으로 비교하는 것을 권장합니다.
